홍익대학교 컴퓨터공학과 소프트웨어공학 수업 과제

github 수업에서 부가적인 기능으로 static website를 구축하는 예를 보았습니다.
Website를 만드는 것은 소공 수업과 직결되는 것은 아니지만 지금의 모든 소프트웨어는 100% 웹 서비스를 구현하는 것이라고 해도 과언이 아닌 만큼 이미 대부분 javaScript 같은 언어로 웹 페이지를 만들어 본 경험이 없는 경우에도 1시간이면 수업 중에 얘기한 github 기능과 front end 웹 페이지 template 사이트에서 제공되는 기본적으로 javaScript 언어로 만들어진 간단한 페이지의 소스코드를 통해 어렵지 않게 공부해 볼 수 있습니다.
과제는 이번 토요일 오후 8시까지 각자 적당한 static 웹 페이지 template를 찾아서 각자의 이름 학번을 페이지에 보이게 하여서 자신의 웹 사이트를 github에 만들어 둔 다음에 github url을 클래스룸 과제로 올려두어 누구나 확인할 수 있도록 하는 것입니다.
